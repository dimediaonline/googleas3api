staticMapURLproperty 
staticMapURL:String  [read-write]

Supplies a url to a static map image representation of the current result. The image is 150px wide by 100px tall with a single marker representing the current location. Expected usage is to hyperlink this image using the url property. The image may be resized using google.search.LocalSearch.resizeStaticMapUrl(). The Static Map Tile sample demonstrates one way to use this property.
