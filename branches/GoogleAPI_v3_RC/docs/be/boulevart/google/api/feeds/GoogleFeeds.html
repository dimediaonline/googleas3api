findFeed()method






public function findFeed(query:String):void

   Find a feed. Will return a list of feeds that match the given query.
   Catch result using EventListener GoogleAPIEvent.FEEDS_FIND_RESULT

   
GoogleAPIEvent.data contains instance of 
GoogleFeedFindResult

   
Example:

      var googleFeed:GoogleFeeds = new GoogleFeeds();
   googleFeed.findFeed("Adobe");
   googleFeed.addEventListener(GoogleAPIEvent.FEEDS_FIND_RESULT,onFeedFindResult);
   

//catch API errors from Google:
   googleFeed.addEventListener(GoogleAPIErrorEvent.API_ERROR, onApiError);
   

   
Parameters




query:String — Keyword(s) to find the feed

numFeeds:int (default = -99) — This optional argument supplies the number of entries to load from the feed specified by feedURL.
A value of -1 indicates the maximum number of entries supported which at the time of this writing is 100.
If this argument is not supplied, a value of 4 is assumed.
   		




 




scoring:String (default = "") — By default, when a feed is loaded, the API returns a cached copy of the specified feed that is 100% in sync with the feed contents at the time that it was cached. By setting scoring to a value of h, you can instruct the system to return any additional historical entries that it might have in its cache.
